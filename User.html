# 备注

# subUrl

TIP

需要配合ssrSubToken使用，组合成为订阅链接，使用方法如下

let subscribeUrl = subUrl + ssrSubToken //构成订阅链接，含有params可选
params 说明
mu 订阅链接类型 0:普通端口ssr | 1:单端口ssr | 2:v2ray | 3: ss/ssd
